# Exporter and VM-Side Baseline Status

## Purpose

This directory contains VM-side and exporter-related baseline materials used to support the private Solana benchmarking workflow.

## Current Status

The exporter and VM-side support path exists in baseline form, but the public packaging of this component is still being systematised.

This component should be read as part of the overall single-node baseline environment rather than as an isolated product.

## Project 1 Role

Project 1 may include:
- documenting the exporter role in the benchmark workflow;
- clarifying how this component interacts with the VM-side environment;
- preparing baseline release materials for public inspection and reproducibility.

## Out of Scope

The current microgrant does not aim to deliver a fully general exporter framework or a later-stage multi-node instrumentation layer.
