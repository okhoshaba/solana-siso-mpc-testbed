# Exporter Research Component Status

This directory is reserved for VM-side latency research support materials
associated with the private Solana benchmarking baseline.

## Current status

This component exists conceptually within the baseline workflow and is being
prepared for clearer public packaging under Project 1.

## Project 1 role

Project 1 may include:
- documenting the role of this component in the baseline workflow;
- clarifying how it relates to exporter and VM-side execution;
- preparing baseline release materials suitable for public review.

## Out of scope

The current microgrant does not attempt to deliver a fully general latency
research framework or a later multi-node instrumentation layer.
