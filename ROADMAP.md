# Roadmap

## Overview

This roadmap separates the current baseline-release phase from later scale-up and research-extension phases.

## Phase A — Baseline Systematisation and Reproducible Release

Scope:
- systematise the repository;
- document the current workflow;
- consolidate baseline host-side and VM-side components;
- automate single-node baseline deployment using Ansible;
- publish a coherent GitHub release;
- prepare a Zenodo archival release.

Expected outputs:
- systematised repository structure;
- baseline deployment automation;
- initial benchmark workflow documentation;
- initial datasets and figures;
- reproducibility package.

## Phase B — Multi-Node Private Solana Cluster

Planned focus:
- extend from single-node to multi-node private Solana deployment;
- introduce controlled topology and network configuration options;
- mature the benchmark workflow for broader experimental campaigns.

Expected outputs:
- multi-node deployment framework;
- larger benchmark scenarios;
- extended observability and experimental coverage.

## Phase C — Adaptive Load-Control Research Extension

Planned focus:
- extend from baseline SISO-oriented experimentation to a broader adaptive control layer;
- introduce initial multi-objective control logic;
- support later MIMO-oriented experimental work.

Expected outputs:
- adaptive-control prototype extensions;
- expanded datasets;
- later-stage technical and research artefacts.

## Status

Current effort is focused on **Phase A** only.
