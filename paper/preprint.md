# Preprint Draft Status

## Working title

Systematised and Reproducible Baseline Benchmarking for a Private Solana Testbed

## Current status

This preprint is in early preparation. The repository already contains baseline
figures, datasets, analysis assets, and release metadata that may later support
a technical manuscript or preprint.

## Project 1 role

Project 1 does not aim to complete a full paper. Its purpose is to:
- systematise the baseline repository;
- document the current workflow;
- automate baseline deployment steps;
- prepare a coherent public release on GitHub and Zenodo.

## Intended later use

The baseline release produced under Project 1 may later support:
- a technical note;
- a preprint;
- later research dissemination associated with follow-on work.

## Out of scope for Project 1

- a complete finished manuscript;
- full multi-node results;
- the full adaptive MIMO research phase.
