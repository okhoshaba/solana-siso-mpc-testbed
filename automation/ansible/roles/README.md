# Ansible Roles Status

This directory is reserved for reusable Ansible roles related to the baseline
single-node private Solana deployment path.

## Current status

Project 1 introduces the public automation structure for baseline deployment.
Reusable roles will be systematised progressively as part of the repository
clean-up and release preparation effort.

## Intended role coverage

Planned role areas may include:
- VM-side baseline preparation;
- host-side baseline support tooling;
- verification and readiness checks.

## Out of scope

Project 1 does not aim to provide a complete multi-node or production-grade
automation framework.
