# GitHub Release Notes — Baseline Release

## Summary

This release provides the first systematised public baseline package for controlled benchmarking in a private single-node Solana environment.

## Included in This Release

- baseline repository structure
- analysis and processing materials
- baseline experimental scripts
- initial datasets and figures
- reproducibility and release metadata
- baseline documentation
- initial automation scaffolding for deployment reproducibility

## Purpose

The purpose of this release is to make the current baseline environment reviewable, reproducible, and suitable for later scale-up work.

## Notes

This release represents the baseline phase only. Multi-node scale-up and later adaptive-control extensions belong to follow-on work.
