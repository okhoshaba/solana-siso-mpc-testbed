# Release Packaging

## Purpose

This directory contains release-related materials for public baseline publication.

## Publication Channels

The project uses two publication channels:

### GitHub
Used for:
- the living repository;
- documentation;
- release notes;
- code and workflow inspection.

### Zenodo
Used for:
- curated archival release packages;
- reproducibility artefacts;
- datasets;
- figures;
- release metadata suitable for citation.

## Project 1 Release Goal

The goal of Project 1 is to prepare a coherent baseline release suitable for:
- technical review;
- public inspection;
- reproducibility support;
- citation and archival reference.

## Planned Materials

- GitHub release notes
- Zenodo release notes
- release metadata
- references to baseline datasets and figures
