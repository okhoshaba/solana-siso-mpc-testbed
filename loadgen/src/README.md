# Load Generator Source Status

This directory is reserved for the host-side load generator source code used in
the private Solana benchmarking baseline.

## Current status

The load-generation component exists in baseline working form and is being
systematised for clearer public release as part of Project 1.

## Project 1 role

Project 1 does not aim to deliver a complete mature benchmarking suite.
Its purpose at this stage is to:
- document the baseline load-generation path;
- prepare the public source layout;
- support later reproducible benchmark execution.

## Out of scope

A broader or more mature benchmark framework belongs to the later follow-on
phase rather than to Project 1.
