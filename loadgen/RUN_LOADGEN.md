# Run Load Generator

## Status

The host-side load generator exists in baseline working form and is being systematised for public release as part of Project 1.

## Purpose

This component is used to generate controlled transaction load for the private Solana benchmarking environment.

## Current Project 1 objective

The current microgrant phase focuses on:
- documenting the baseline load-generation path;
- clarifying execution inputs and outputs;
- preparing the initial public release materials.

## Planned usage path

1. prepare the host environment;
2. confirm connectivity to the baseline private Solana node;
3. select a baseline scenario;
4. execute the load generator;
5. collect raw outputs for later processing.

## Note

A fuller benchmark suite is planned for the later follow-on phase rather than for Project 1.
